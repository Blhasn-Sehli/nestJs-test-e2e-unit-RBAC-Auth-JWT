
## Description

######

Basic Role Based Access (RBAC) Controll With Nest js  With Unit Test && e2e test  Implementation 

## Used Techs

- Nestjs
- Mongoose ODM
- bcrypt
- jwt

## Installation

```bash
npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

```

## Test

```bash
# unit tests
$ npm run test

#watch mode
$ npm run test:watch

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov

```
